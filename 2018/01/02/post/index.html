pptp 하는 법

  1. pptp 설치
    yum install ppp

wget http://poptop.sourceforge.net/yum/beta/packages/pptpd-1.4.0-1.rhel5.1.x86_64.rpm
rpm -Uvh pptpd-1.4.0-1.rhel5.1.x86_64.rpm

  1. 설정파일 수정
    /etc/pptpd.conf < 파일 맨아래

localip 10.0.0.1 <<쓰고 싶은ip
remoteip 10.0.0.2-50 << 쓰고싶은 ip

/etc/ppp/options.pptpd

#require-mppe-128 << 주석처리 (안그럼 url 접속안됨)

ms-dns 8.8.8.8 <dns서버 설정
ms-dns 164.124.101.2

  1. 계정설정

/etc/ppp/chap-secrets

(ID) (VPN서버) (패스워드)        (client에 할당할 ip 지정해주어도 되고 *로 놓아도됨 , 지정 해주면 remoteip 에 있는 ip 작성)
  1. iptables에서 nat 설정

pptpd.conf 파일에 설정 해놓은 사설ip (ex 10.0.0.0) 대역을 써줍니다
/etc/sysconfig/iptables <파일에 아래와 같이 작성해주어도 되고

iptables에서 pptp 포트 오픈(아래 포트외에도 vpn client에서 사용할 포트는 vpn 서버에서 열려있어야 됩니다)

-A RH-Firewall-1-INPUT -p tcp -m state –state NEW -m tcp –dport 1723 -j ACCEPT
-A RH-Firewall-1-INPUT -p tcp -m state –state NEW -m tcp –dport 47 -j ACCEPT

pptp에서 gre 사용하도록 모듈추가 (pptp재시작후 아래 두명령어 꼭 입력해야됩니다!)
modprobe ip_nat_pptp
modprobe ip_conntrack_pptp
